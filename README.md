# request-extensions

本库是对请求（axios、fetch）常见问题处理方案的封装，包括取消请求、重复提交请求、缓存请求等。

## 安装

```
yarn add request-extensions
```
## api

### axios

#### 取消请求

- cancelableAxios
- cancelAllAxios
- isCancel

#### 重复提交请求
- notRepeatableAxios
- deleteRequestingAxios
- isRepeatSubmit

#### 缓存请求

- cacheableAxios


### fetch

#### 取消请求
- cancelableFetch
- cancelAllFetch
- isCancel

#### 重复提交请求

- notRepeatableFetch
- deleteRequestingFetch
- isRepeatSubmit

#### 缓存请求

- cacheableFetch
